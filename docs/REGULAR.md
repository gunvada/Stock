# 본장 트랙 — 눌림목(흡수) 반등

> 이 트랙만 관리하는 Claude Code 세션은 이 문서를 먼저 읽고 시작하세요.

## 전략 요약
- **셋업**: 거래량 ≥10배 폭증 + 당일 −5~−15% 음봉(흡수) + $0.3~20 + 거래대금 ≥$2M + 직전2일 누적 <+100%(파라볼릭 제외).
- **타이밍**: 신호일 D(전일 마감) → **매매일 D+1**. 다음날 시초가 부근 진입(VWAP 회복+거래량 확인) → **+10% 익절 / −8% 손절 / 종가 전 청산(오버나잇 금지)**.
- ⚠️ **청산이 미국 종가(16:00 ET = KST 05:00)** 라 **회원 매매시간 밖**입니다. 자동 브래킷(OCO) 주문 전제로만 현실적. (프리마켓 트랙과 결이 다른 핵심 이유)

## 파일·파이프라인
| 역할 | 파일 |
|------|------|
| 후보 추출 | `pullback_scanner.py` (← `scanner.py` Polygon grouped) |
| 자동 채점 | `daily_verify.py` |
| 확대 백테스트 | `pullback_backtest.py` |
| 워크플로 | `.github/workflows/daily.yml` (22:00 UTC = KST 07:00) |
| 산출물 | `output/pullback_<날짜>.csv` |
| 성적 장부 | `output/verification_ledger.csv` |

## 운영
- 매 평일 22:00 UTC 자동 실행 → 오늘 픽 생성 + 전일 픽 채점.
- 빠른 확인: `python dashboard.py regular`
- 데이터: Polygon (무료 티어는 마감 후 몇 시간 지연 → 새벽/아침 KST에 갱신).

## 세션에서 할 일 (관리 루틴)
1. `python dashboard.py regular` 로 최신 픽·누적 성적 확인.
2. 채점 결과가 이상하면 Actions 로그/`verification_ledger.csv` 점검.
3. 개선 작업은 ROADMAP의 `[본장]` 항목부터.

## 이 트랙의 로드맵 (→ [docs/ROADMAP.md](ROADMAP.md))
- **P0**: 워크포워드 검증 · 리버스스플릿 필터 · 분봉 백테스트 · (공통) 비용현실화 · 사이징.
- **P1**: 데이터 교차검증 확대.

## 알려진 한계
- 엣지가 **꼬리 의존**(중앙값≈0): 소수 대박이 평균을 만듦 → 손절 규칙 준수가 생존 핵심.
- 60일 검증서 승률 48%·플랜 손익 ≈본전 → **실투자 전 워크포워드 필수**.
- 청산 시간이 회원 시간 밖 → 브래킷 주문 없으면 첫1시간 투매만 떠안음(실측 −6.7%).
