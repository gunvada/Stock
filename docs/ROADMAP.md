## P0 — 실투자 전 필수

- [ ] **표본·기간 확대 + 워크포워드 검증** `[본장]` `[프리마켓]`
  - 현재 60일/단일 장세 → 1~2년치, 앞구간 튜닝→뒷구간 검증으로 과적합 제거.
  - 근거: 15일 승률 60% → 60일 48% 하락(과적합 신호).
- [ ] **거래비용·슬리피지 현실화** `[공통]`
  - `COST=2.5%` 고정 → 가격대·거래대금별 차등 스프레드/슬리피지, 갭하락 손절 슬립 반영.
- [ ] **리버스 스플릿 탐지·제외 필터** `[본장]`
  - 역분할이 가짜 거래량배율/가짜 급등을 만듦(페니주 빈번). 분할 이벤트 종목 제외/플래그.
- [ ] **분봉 백테스트(룩어헤드/체결 정밀화)** `[본장]` `[프리마켓]`
  - TP·SL 같은 날 둘다 터치 시 보수적 손절가정 → 분봉으로 선후 판정. 실제 진입룰(VWAP회복) 재현.
- [ ] **포지션 사이징·리스크 관리 도입** `[공통]`
  - 현재 자금관리 규칙 전무. 트레이드당 리스크%, 동시보유 한도, 일일 손실컷, MDD·파산확률, 켈리 사이징.
